# 🔥 MAC Shifter Pro

> **Professional MAC Address Manipulation Tool**

---

## 📌 Overview

**MAC Shifter Pro** is a Python-based tool designed to change, randomize, and manage MAC addresses efficiently using multiple advanced techniques.

---

## 🚀 Features

- Display current MAC address  
- Modify device-specific identifier (last 3 bytes)  
- Generate random MAC (same vendor)  
- Generate random MAC (different vendor)  
- Restore original MAC address  
- Fully random MAC generation  
- Factory-like MAC address generation  
- Manual MAC address assignment  
- Continuous MAC rotation (automated mode)  

---

## 🛠️ Requirements

- Python 3.x  
- Linux-based OS (Kali Linux / Ubuntu recommended)  
- macchanger installed  
- Root privileges  

---

## ⚙️ Installation

```bash
git clone https://github.com/D4rkScyth3/MACShifterPro.git
cd mac-shifter-pro
pip install -r requirements.txt
python msp.py
```

---

## 🧠 Working Principle

The tool uses Python's subprocess module to execute system-level commands via macchanger.

It enables:
- MAC spoofing  
- Vendor manipulation  
- Automated MAC rotation  
- Interface-based control
